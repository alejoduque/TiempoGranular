To run:
- install nodejs
- download repository
- enter repository folder (from terminal):
```cd ckweb-web```
- install dependencies
```npm install .```
- start dev server
```npm run start```

To build:
```npm run build```

To deploy to github pages:
```npm run deploy```

To merge gh-pages branch to master:
```git merge origin/gh-pages master --allow-unrelated-histories```
